# Asset Report Generator
A simple pipeline automation tool written in Python that scans project folders and generates asset reports.

## Features
- Scans project folders
- Counts the number of files
- Counts file types
- Generates asset_report.txt
- Useful for pipeline and workflow management

## How to use
Run the script:
```
python main.py
```

Enter the folder path, and the script will generate an asset report.

## Example Report
```
Asset Report
====================
Total Files: 120

File Types:
.png: 45
.jpg: 20
.exr: 30
.mov: 10
.txt: 15
```

## Purpose
This project was created to practice pipeline tools development, file management and workflow automation for VFX and real-time production environments.
